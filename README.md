# International Credit

International Credit is a full-stack platform comprising:

- **International-credit-be** – Node.js backend (double-entry ledger, Stripe, Sumsub KYC, audit trails)
- **International-credit-admin-fe** – Admin frontend
- **International-credit-user-fe** – User frontend
- **International-Credit-SC** – Smart contracts

See each subdirectory for setup and documentation.
